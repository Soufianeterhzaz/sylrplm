application sylrplm : Product Life Management

  • Description sommaire 
          - Objet:gestion de produits et de leurs cycles de vie
      - Données: documents, articles, projets, clients
      - Cycle de vie: maturité, révision, checkin/checkout
      - Paramétrage: type, status, valeurs par défaut ...
      - Volumes de stockage de fichiers.
  • pre requis 
          - mysql
  • mise en place 
          - décompresser le fichier zip ou charger l'application depuis github:

      - se placer sur le repertoire principal de l'application
              cd sylrplm
  • installation des plugins (facultatif je pense) 
          - plugin de gestion d'autorisations
              ruby script/plugin install http://opensvn.csie.org/ezra/rails/plugins/dev/acl_system2
  • implémentation de l‘application 
          - créer la base
              mysqladmin -u root create sylrplm_production

      - charger le schema
              rake db:schema:load RAILS_ENV=production
